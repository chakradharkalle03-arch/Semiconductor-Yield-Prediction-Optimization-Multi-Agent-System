GitHub Actions workflows are already set up in `.github/workflows/`. They will automatically run on:
- Every push to main/master branch
- Every pull request
- When documentation files are updated

To view workflow runs:
1. Go to the "Actions" tab in your repository
2. You'll see workflow runs and their status
